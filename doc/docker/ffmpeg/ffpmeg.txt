1.拉取镜像
docker pull linuxserver/ffmpeg

2. 启动容器
docker run -itd --name ffmpeg \
-p 8066:8066 -v /Users/garlam/ace/videos/:/tmp/ \
--entrypoint='bash' linuxserver/ffmpeg

3.不影射文件夹
docker run -itd --name ffmpeg \
-p 8066:8066 \
--entrypoint='bash' linuxserver/ffmpeg

--entrypoint='bash' =>bash指的是容器里的bash

4.进入容器
docker exec -it ffmpeg /bin/bash

5.检查ffmpeg版本
ffmpeg -version

6.宿主机执行ffmpeg容器命令
docker exec -it ffmpeg ffmpeg -version

第一个ffmpeg是容器名称
第二个ffmpeg是命令
