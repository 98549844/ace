docker安装mysql，自定义配置文件

docker run \
--name mysqlname \
-p 3306:3306 \
-v /tmp/mysql/conf/hmy.conf:/etc/mysql/conf.d/hmy.conf \
-v /tmp/mysql/data:/var/lib/mysql \
-e MYSQL_ROOT_PASSWORD=test123 \
-d mysql:8.0.29

–name 设置容器运行名称
-p 设置端口号
-v 映射配置文件地址 /tmp/mysql/conf/hmy.conf 是我的配置文件目录 映射到 /etc/mysql/conf.d/hmy.conf
-v 映射配置文件地址 /tmp/mysql/data 是我的配置data目录 映射到 /var/lib/mysql
-e MYSQL_ROOT_PASSWORD=test123 设置mysql密码是test123
-d 镜像+版本号
