搜寻可用的mysql版本
docker search mysql

1. pull mysql
docker pull mysql =>拉取mysql最新版本
docker pull mysql 8.xx =>拉取mysql指定版本

查看本地镜像
docker images

2.运行容器
=>  docker run -itd --name mysql-8.0 -p 3306:3306 -e MYSQL_ROOT_PASSWORD=root mysql
=>  docker run -itd --name mysql mysql:latest -p 3306:3306 -e MYSQL_ROOT_PASSWORD=root

=>  docker run -it --name mysqlserver -p 3306:3306 -e MYSQL_ROOT_PASSWORD=root
    --network mynet --network-alias mysqlserver a3a2968869cf080dbbd2adaac9e4075cc358b50a1451ff5e2b9ae90551a4735f

=>  docker run -d -p 3306:3306 -v /home/mysql/conf:/etc/mysql/conf.d -v/home/mysql/data:/var/lib/mysql
    -e MYSQL_ROOT_PASSWORD=root --name mysql0 mysql:5.7


参数说明:
-p 3306:3306: 映射容器服务的 3306 端口到宿主机的 3306 端口, 外部主机可以直接通过 宿主机ip:3306 访问到 MySQL 的服务
MYSQL_ROOT_PASSWORD=root: 设置 MySQL 服务 root 用户的密码

3. access mysql console
mysql -h localhost -u root -p

4.show version
select version();

5.进入mysql容器shell
docker exec -it 容器的id /bin/bash
docker exec -it mysql-8.0 /bin/bash
