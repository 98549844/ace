1.运行容器
docker run -d --name activemq -p 61616:61616 -p 8161:8161 apache/activemq-classic:latest

解释命令：

-d：表示以后台模式运行容器。
--name activemq：给容器指定一个名称，方便后续管理。
-p 61616:61616：将主机的61616端口映射到容器的61616端口，用于消息传输。
-p 8161:8161：将主机的8161端口映射到容器的8161端口，用于Web管理界面。
运行完以上命令后，ActiveMQ容器将在后台运行，并监听主机的61616和8161端口。
