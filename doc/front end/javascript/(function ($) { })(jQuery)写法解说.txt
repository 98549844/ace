(function ($) {
    /* 代码块 */
})(jQuery);

称为立即执行函数表达式
(Immediately Invoked Function Expression, IIFE)

IIFE是一种将函数包装在括号中并立即调用的技术. 
在这种情况下, 
函数接受一个参数 $, 它被用作函数内部的局部变量. 
通常, 
这个参数用于避免与全局作用域中的其他可能存在的全局变量或标识符冲突. 

(jQuery)则是将jQuery库作为参数传递给立即执行函数. 
通过将jQuery作为参数传递给函数, 
并在函数内部将其赋值给$变量, 
可以确保在函数内部使用$时, 
它指向的是jQuery库的别名

将jQuery库作为参数传递给IIFE的好处是可以在函数内部使用$作为对jQuery库的引用, 
同时避免与全局作用域中的其他变量冲突. 
这是一种常见的用法, 特别是当代码中同时使用多个JavaScript库时

总结起来, (function ($) { /* 代码块 */ })(jQuery);
这种写法是一个立即执行函数表达式(IIFE), 
通过将jQuery作为参数传递给函数, 
并在函数内部将其赋值给$变量, 
可以确保在函数内部使用$时, 它指向的是jQuery库的别名
