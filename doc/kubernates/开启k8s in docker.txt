1. 配置安装控制台 kubernetes dashboard
kubectl apply -f https://raw.githubusercontent.com/kubernetes/dashboard/v2.7.0/aio/deploy/recommended.yaml

更多资料: https://kubernetes.io/zh-cn/docs/tasks/access-application-cluster/web-ui-dashboard/


2. 开启 API Server 访问代理 默认监听 127.0.0.1:8001 端口
kubectl proxy #一直占着终端,可以用下面命令
nohup kubectl proxy >/dev/null &


3. 通过如下 URL 访问 Kubernetes dashboard
http://localhost:8001/api/v1/namespaces/kubernetes-dashboard/services/https:kubernetes-dashboard:/proxy/
http://localhost:8001/api/v1/namespaces/kubernetes-dashboard/services/https:kubernetes-dashboard:/proxy/#/login
选择 kubeconfig

4. 创建 ServiceAccount kubernetes-dashboard-admin 并绑定集群管理员权限
创建文件 kube-system-default.yml
kubectl apply -f kube-system-default.yaml


5. 获取登陆 token
kubectl -n kube-system describe secret default| awk '$1=="token:"{print $2}'
