键盘背光调节
FN + HOME:      RGB灯效切换
                波浪模式  龙飛凤舞  激光模式
                涟渏模式  斜拉变换  跑马模式
                正弦波    瀑布模式  交错跑马
                随按即亮  随安即灭  繁星点点
                落雨模式  漫天模式  单色呼吸
                七彩渐变  电波扩散

FN + DEL:       RGB灯效颜色(单色+RGB)切换
FN + INS:       循环调节常亮颜色
                (默认RED=>GREEN=>BLUE=>YELLOW=>PURPLE=>SKY BULE=>ORANGE=>WHITE=>OFF)
FN + END:       循环调节常亮颜色
FN + SPACE BAR: LED效果暂停/启动
FN + ⇅:         调节LED亮度
FN + ⇆:         灯效速度增加/减少

区域灯效显示
FN + 8:         区域灯效显示  ESC 123456 QWER ASDFB CTRL-L ALT-L
FN + 9:         区域灯效显示  ESC WASDR 方向键
FN + 0:         区域灯效显示  QWERTYUIOP ASDFGHJKL ZXCVBNM

自定义灯光录制
FN + backspace: 录制和保存自定义模式
FN + -_:        自定义模式1, 默认:空
FN + =+:        自定义模式2, 默认:空
选择以上的两组灯光的任意一种,按FN + BACK进入灯光自定义录制功能
同时功能按键-_ / +=灯处于闪烁状态,键入需要录制的灯光,
挗键重复按下可选8色+关闭,录制完成后按FN + BACK保存录制退出
指示状态恢复

F2:     音量减小
