1.  首先下载nvm
    下载地址：https://github.com/coreybutler/nvm-windows/releases
    或者 https://download.csdn.net/download/xiaofiy/85235297
    下载安装过程建议傻瓜式安装，直接安装到c盘

    mac version nvm
    terminal access (raw.gitmirror)
    curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
    curl -o- https://raw.gitmirror.com/nvm-sh/nvm/v0.39.6/install.sh | bash
    curl -o- https://usercontent.githubfast.com/raw/nvm-sh/nvm/v0.39.6/install.sh | bash
    https://githubfast.com/nvm-sh/nvm#install--update-script


1.1 检查nvm版本
    nvm --version

2.  列出可供安装的所有 Node.js 版本
    nvm ls-remote


2.1  安装14版本和16版本的node（默认会安装最新最稳定版本）
    nvm install 14
    nvm install 16

3.  查看nvm路径命令
    nvm root

4.  列出所有可管理的node版本
    nvm ls

5.  切换使用node时,
    只需nvm use 版本号，指定当前使用的node版本
    nvm use node_version

6.  删除 Node.js 版本
    nvm uninstall <version>
    nvm uninstall 16.20.0

7. 解除当前已激活的nvm版本
   nvm deactivate

8. 查看当前版本
   nvm current

9. 设置默认node版本
   nvm alias default 18.19.0

9.1 取消默认版本设置
   nvm unalias default

9.2 验证默认版本
   nvm alias
