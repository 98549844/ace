
springboot2和3的区别
Spring Boot 2和3之间存在多个主要区别：

最低环境要求：
Spring Boot 2的最低版本要求为Java 8，并支持Java 9。
Spring Boot 3的最低版本要求为Java 11，并且支持Java 17及更高版本。12
Java虚拟机 (JVM) 支持：

Spring Boot 2基于Java 8，而Spring Boot 3基于Java 11及以上版本，这允许使用更高级的语言特性和更高效的JIT编译器。
Spring框架版本：

Spring Boot 2使用的是Spring Framework 5。
Spring Boot 3构建于Spring Framework 6之上，提供了更好的性能和功能。13
第三方库支持：

Spring Boot 3删除了对一些旧版本的依赖项支持，如Apache ActiveMQ、Atomikos、EhCache 2和Hazelcast 3。
响应式编程支持：

Spring Boot 3更加注重响应式编程范式，提供了更多与响应式相关的功能和支持。4
Web框架支持：
Spring Boot 2默认使用Spring MVC作为Web框架。
Spring Boot 3将引入对Spring WebFlux的支持，这是一个非阻塞的、响应式的Web框架，适合构建高性能的异步和事件驱动的应用程序。2
其他变化：
Spring Boot 3还包括了一些其他的特性，如Java EE变更为Jakarta EE、Log4j2增强、三方包升级等。3
