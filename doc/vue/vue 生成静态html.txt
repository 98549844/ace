1.  通过 Vue CLI 创建一个新项目
    vue create my-project

2.  进入到 my-project 文件夹中, 安装 Vue 的静态网站生成工具(Vuepress)
    npm install -D vuepress

3.  创建一个 markdown 文件(README.md),如果没有

4.  创建一个 .vuepress 文件夹, 并在其中创建 config.js 文件
    其中, nav 和 sidebar 是主题页面导航的配置, 可以根据自己的需要进行修改
    module.exports = {
        title: 'My Site', // 网站的标题
        description: 'This is my site', // 网站的描述
        themeConfig: { // 主题配置
            nav: [
                {text: 'Home', link: '/'},
                {text: 'Guide', link: '/guide/'},
                {text: 'External', link: 'https://google.com'},
            ],
            sidebar: [
                '/',
                '/guide/',
            ]
        }
    }

5.  通过命令在本地启动网站
    npx vuepress dev

6.  执行命令构建生成静态 html 文件
    npx vuepress build

之后静态 html 文件就会生成在 .vuepress/dist 目录下了.
将这个目录下的文件上传到服务器上即可.

通过以上步骤, 我们就可以成功生成 Vue 的静态 html 文件了.
与直接在网页中渲染不同, 这种方式生成的页面在搜索引擎中更容易被查找到,
也更加方便被搜索引擎爬虫抓取. 此外, 生成静态 html 页面的请求响应速度也会更快, 从而提升了用户的体验.
