项目名: ace-firebase-application
web应用: npm install firebase

firebase安装: npm install firebase
firebase工具: npm install firebase-tools


cli登入: firebase login
server site logon: firebase --no-localhost

初始化:
npm init -y (build empty nodejs project)
-y: all option select Yes

布署: firebase deploy

list all project
firebase projects:list
